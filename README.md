## YouTube Ad Skipper (For Falkon)
A Falkon extension that skips (not block!) YouTube ads.  
Works on my machine

## Installation
1. Download the file from the Release tab
2. Extract it into a folder
3. Put it in `~/.config/falkon/plugins/`
4. Restart Falkon
5. Stonks
